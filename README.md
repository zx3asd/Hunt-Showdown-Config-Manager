# Hunt-Showdown-Config-Manager
Advanced configuration tool for Hunt: Showdown: dual binds, custom FPS lock, HUD scaling, and profile management.

## English

A tool for advanced Hunt: Showdown configuration that allows you to modify settings not available through the standard in-game menu.

### Key Features

  * **Dual Binds:** Assign two different keys to the same in-game action.
  * **Custom FPS Lock:** Set a custom frame rate limit beyond the game's standard presets.
  * **HUD Scaling:** Adjust the size of interface elements for optimal visibility.
  * **Precise Sensitivity Tuning:** Fine-tune your sensitivity with higher precision for better aiming.
  * **Profile System:** Create multiple configuration profiles and swap between them instantly.

## Russian / Русская версия

Программа для тонкой настройки конфигурации Hunt: Showdown, которая позволяет изменять параметры, недоступные через стандартное игровое меню.

### Основные возможности

  * **Двойные бинды:** возможность назначить две разные клавиши на одно и то же игровое действие.
  * **Кастомный FPS lock:** установка произвольного ограничения частоты кадров, отличного от стандартных пресетов игры.
  * **Масштабирование HUD:** изменение размеров элементов интерфейса для настройки оптимального обзора.
  * **Точная настройка чувствительности:** регулировка сенсы с более высоким шагом точности для идеального наведения.
  * **Система профилей:** возможность создавать несколько разных конфигураций и мгновенно переключаться между ними.
